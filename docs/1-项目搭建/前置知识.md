# 前置知识

在开始本项目前，需要具备以下前端基础：

## 1. ES6+ 语法

- 箭头函数、解构赋值、模板字符串
- 模块化（import/export）
- Promise、async/await
- 类与继承
- 扩展运算符、剩余参数

## 2. TypeScript 基础

- 类型注解与类型推断
- 接口与类型别名
- 泛型
- 类与装饰器
- 模块声明

## 3. 前端工程化

- 包管理器使用（npm/pnpm）
- 项目构建流程
- 调试与热重载
- 代码打包与优化
- 部署流程

## 4. 开发工具

- Git 版本控制
- 代码编辑器（VS Code 等）
- 浏览器开发者工具
- 终端命令行操作

## 项目结构说明

本项目需要构建三个主要部分：

### 核心库 (lib)
使用 TypeScript 编写的 GIS 引擎，包含地图、图层、几何、符号等核心功能。

### 演示示例 (demo)
展示库功能的示例页面，包含各种图层类型的实际应用。

### 文档站点 (docs)
基于 VitePress 的技术文档，提供开发指南和 API 参考。
