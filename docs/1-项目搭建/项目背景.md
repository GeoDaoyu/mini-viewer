# 背景

作者是一名 WebGIS 从业者，曾在 2019 年至 2021 年间断断续续推出了[《ArcGIS JS API学习之旅》](https://space.bilibili.com/6100019/lists/1399871?type=season)系列视频。但时至今日，视频内容大多已跟不上版本更新的步伐。

有一种观点认为，学会一门技术的最好方式之一，就是亲手实现它。此前也曾涌现过不少像 `mini-react` 和 `my-promise` 这样的前端实现教程。

如今正值 Vibe Coding 时代，于是萌生了一个想法：借助 AI 来实现一个 mini 版的 ArcGIS JS API。
