# 技术栈

## 构建工具

本项目使用 **[Vite](https://vite.dev/)** 作为构建工具，提供以下优势：

### 开发体验
- 极速的冷启动
- 快速的热模块替换（HMR）
- 按需编译，无需打包整个应用

### 生产构建
- 优化的 Rollup 打包
- 代码分割和懒加载
- 预构建依赖

### 生态系统
- 原生支持 TypeScript
- 丰富的插件系统
- 与主流框架无缝集成

## 核心依赖

### 开发依赖
- **TypeScript** - 类型安全的 JavaScript 超集
- **Vite** - 现代前端构建工具
- **VitePress** - 基于 Vite 的静态站点生成器

### 运行时依赖
- **@geodaoyu/accessor** - 属性访问模式支持

## 项目配置

### 构建配置
- **主构建** - `vite.config.ts`，入口为 `./lib/main.ts`
- **演示构建** - `vite.config.demo.ts`，输出到 `demo/` 目录

### 输出格式
- ESM 模块（现代浏览器和打包工具）
- UMD 模块（传统浏览器和直接使用）

## 开发命令

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm start

# 构建库源代码
pnpm run build

# 构建演示版本
pnpm run demo
```
